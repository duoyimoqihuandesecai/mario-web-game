# Mario Web Game

这是一个纯前端静态网页小游戏，已经整理成可直接发布到 GitHub Pages 的在线版。

## 项目文件

- `index.html`
- `styles.css`
- `game.js`
- `.nojekyll`


## 发布到网上

推荐使用 GitHub Pages。步骤如下：

1. 在 GitHub 新建一个仓库，比如 `mario-web-game`
2. 把当前文件夹里的文件上传到仓库根目录
3. 确保默认分支名是 `main`
4. 进入 GitHub 仓库页面
5. 打开 `Settings` -> `Pages`
6. 在 `Build and deployment` 中选择从 `main` 分支和根目录发布
7. 等待 GitHub Pages 完成部署

发布成功后，公网网址通常是：

```text
https://你的GitHub用户名.github.io/mario-web-game/
```

例如用户名是 `zhangsan`，仓库名是 `mario-web-game`，网址就是：

```text
https://zhangsan.github.io/mario-web-game/
```

## Edge 打开

发布完成后，任何电脑上的 Edge 浏览器都可以直接打开这个公网网址。

## 游戏说明

- 左右方向键移动
- `Space` 跳跃
- 收集金币
- 躲开尖刺和敌人
- 收集至少 6 枚金币后触碰终点旗帜即可通关
